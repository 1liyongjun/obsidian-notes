# 🧠 Obsidian 笔记仓库

> 我的第二大脑 - 收集整理优秀文章、知识笔记和深度研究

## 📁 目录结构

```
├── 00-Inbox/          # 收件箱 - 临时存放待整理的内容
├── 01-AI-ML/          # AI & 机器学习
├── 02-Programming/   # 编程技术
├── 03-Products/       # 产品 & 设计
├── 04-Linux/          # Linux & 系统运维
├── 05-Reading/        # 阅读笔记
├── 06-Vaults/         # 其他知识库
└── .obsidian/         # Obsidian 配置
```

## 📝 使用说明

### 添加新笔记

1. 将文章链接或摘要放入 `00-Inbox/`
2. 让我帮你总结或补充
3. 整理后移动到对应分类目录

### 让我帮你收集整理

发送给我：
- 文章链接 → 我总结存入对应目录
- 主题 → 我深度研究后存入
- 定时任务 → 自动监控并整理优秀内容

